# SwitchOn Assignment

> Backend API for application, frontend in for client side

## Usage
##to run backend
Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run frontend

#open frontend folder

npm install

npm start

```
